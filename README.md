# Broadside

Next.js landing site for the Broadside sci-fi FPS-Z project.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run start
```

## Deployment

- Push the repo to GitHub.
- Connect the repository to Vercel.
- Use the default Next.js framework preset.
- The media used by the site lives in `public/media`.

## Content Notes

- The hero area auto-plays local gameplay clips.
